Lektion vecka 5

Hemma

Det är även värt att notera att jag mellan förra lektion och denna blev nyfiken på hur man kunde implementera en "paralax effekt" med en bakgrunds bild som scrollar i annan hastighet än sidan. Dock kunda jag bara hitta exempel som använde jquery, så jag letade efter info om hur man kunde hitta scroll nivån med javascript och använde sedan detta för att med dom byta plats på min bakgrund och på så sätt uppnå en parallax effekt. Detta funkade inte särskilt bra i början men sedan insedde jag att om jag ändrade till position fixed så kommer inte elementet påverkas av den vanliga skrollen och bara av min paralax effekt och efter jag ändrat det så såg min sida mycket bättre ut.

På lektion

Idag så satte jag in mer på start och about sidan, jag skapade nav bars och main sektioner efter min skiss. Just nu fattas bara hamburgar menyns funktionalitet, jag bytte även bakgrunds bild.

Lektion vecka 6

Hemma

Jag märkte att min parallax funktion inte längre fungerade och kollade på en gammal commit för att se vad som hade ändrats, det som hade ändrats var att jag hade bytt background från ett id till en klass och märkte då att mitt bakgrunds element hade två class= och inte båda klasser i samma class=, efter att ändrat detta funkade sidan igen

På lektion

Till att börja med idag fixade jag ett problem med min footers längd genom att ta bort min width rad och byta ut den mot en max width på 100vw, på så sätt så overflowar den inte och låter en scrolla i sidled. Jag hade även ett problem med grids där jag hade skrivit in mina grid areas fel men använde style editorn i firefox för att experimentera få reda på vilka rader som var fel.

Det jag har lagt till denna lektion är en sida med produkt listing och börjat på en sida för individuella produkter. Något som jag gjorde som jag tyckte var skoj var att jag använda klasser på min produkt listing sida för att få alla kort att ha ett grid som ser likadant ut men där man kan ändra innehållet. Jag märkte även lite olika problem med min body styling och annat mellan olika sidor och har nu använt mig utav olika klasser för att se till att alla element är där de ska på alla sidor.

En liten reflektion är att jag har använt mycket position absolute på grund av hur jag gjort min bakgrund och att deta nog ledit till att strukturen av min sida blivit lite snurrigt

På lektion

Jag har gjort en cart sida genom att modifiera min listing sida och ändra på hur saker läggs ut, jag planerar att i framtiden lära mig om cookies och js för att se till att carten funkar om jag har tid över, det hade även varit kul att skapa en checkout sida, men jag vet inte om jag har tid med det så det är inte en prioritet för mig just nu.

Lektion vecka 9

Hemma

På lektion

Jag hade ett problem med att bilder inte funkade som jag ville att de skulle, antingen stretchade de fel eller så overflowade det ut över bakgrunden. Lektion vecka 10

Hemma

På lektion

Jag började idag använda mig av css media queries för att ändra på css för desktop sidan så att allt funkar. Jag hade lite problem med att ändra mitt grid då det overflowade och top upp en massa plats på botten av sidan. Anledningen till detta var att min media query inte skrev över grid-template-rows attributen vilket betyde att gridet tog up plats för en massa rows som inte fanns. Mycket av lektionen spenderades dock med att försöka förstå varför det overflowade och jag använde mig av firefox's inspect tool för att se vilka element som var för stora.

Ett annat problem som kom upp var att mina "kort" som var grid children stretchades, detta var för att jag hade skrivit mina rows som fractions och inte den storlek jag ville ha.
